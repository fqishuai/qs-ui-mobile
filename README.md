```bash

# or
$ yarn add qs-ui-mobile
# or
$ pnpm add qs-ui-mobile
# or
$ bun add qs-ui-mobile
```

- 无需配置，即可拥有极致性能。

- 参考[ant-design-mobile](https://github.com/ant-design/ant-design-mobile)和[uView](https://github.com/umicro/uView2.0)中组件的外观及交互，助力打造极致体验的产品。
